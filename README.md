#  Thriftify

Your modern marketplace for discovering, buying, and selling pre-loved fashion with confidence.

##  Description

Thriftify is a full-stack marketplace designed to make second-hand shopping simple, accessible, and enjoyable. The platform connects buyers with sellers through an intuitive interface where users can browse unique fashion items, manage listings, communicate securely, and discover affordable styles while promoting sustainable shopping habits.

## Problem

Buying and selling second-hand items can often be frustrating due to:

- Scattered listings across multiple platforms
- Poor product organization and search experiences
- Limited communication between buyers and sellers
- Difficulty managing personal listings
- Lack of a dedicated, modern marketplace focused on simplicity and trust

Thriftify addresses these challenges by providing a centralized platform with a clean user experience and intuitive marketplace features.

## Features

-  Search and filter product listings
-  Browse items by category
-  Upload product images
-  User authentication and authorization
-  Create, edit, and manage listings
-  Save or favorite items (if implemented)
-  Responsive design

## 🚀 Getting Started

```bash
# Clone the repository
git clone <repository-url>

# Install frontend dependencies
cd client
npm install

# Install backend dependencies
cd ../server
npm install

# Start the development servers
npm run dev
```
